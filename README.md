## ColoryrServer  
在线动态编译.多功能.服务器框架  
**服务器框架内不包含任何业务代码，需要用户自行编写**

![截图](./doc/pic/work.png)

- 这是一个中型服务器
- 支持Linux和Windows下运行
- 可以对接Mysql\Redis\MsSql\Oracle数据库
- 自带Http\WebSocket\Socket\Mqtt支持
- 可以添加ssl证书(ASP模式)
- 可以对接[ColorMirai](https://github.com/Coloryr/ColorMirai) QQ机器人
- 占用内存极少
- 业务代码修改无需重启
